“## מטרת הפרויקט
מטרת הפרויקט היא לנתח נתוני תשלומים של מומחים ובוחנים בין השנים 2018–2025,  
במטרה לזהות מגמות, לבדוק איכות נתונים, ולאתר תשלומים עם שגיאות.  
הפרויקט מדגים שימוש בספריות Pandas, Matplotlib ו־Seaborn  
לייבוא, ניקוי, ניתוח וויזואליזציה של נתונים לצורך הסקת מסקנות מבוססות נתונים.
